*********************************************************
Aplicacion Web: Soluciones Integrales SL
*********************************************************

Esta aplicación utiliza una arquitectura Model-Vista-Controlador y cuenta con un sistema de autenticación.

Lenguaje PHP, HTML, JavaScript, CSS

Se conecta con la base de datos para poder obtener y manipular la info almacenada.

Al iniciar sesion el usuario puede tener privilegio de ADMINISTRADOR, o privilegio de USUARIO.
De acuerdo al privilegio de la sesion se mostrarán las opciones correspondientes.

Los usuarios con privilegio de administradores puede eliminar registros de la base de datos, y administrar los usuarios (añadir, eliminar, visualizar).
